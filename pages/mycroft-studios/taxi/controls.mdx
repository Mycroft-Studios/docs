# Controls

The Taxi System is incredibly easy and simple to use, as well as having all the controls completely customisable!

## Commands

- `/calltaxi` - by default, this command is enabled and will allow any player to call their current position.

## Default Controls

- `Change Vehicle View`: V
- `Toggle Meter Interation`: M
- `Hurry Mode`: X

### Destination Controls

- `Next Destination`: Right Arrow
- `Previous Destination`: Left Arrow
- `Select Destination`: Enter
- `Refresh Destinations`: X
